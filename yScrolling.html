jes.yScrolling

var oScroll = new jes.yScrolling(sCid, iMillisec1,iMillisec2, iSpeed[,iHeight])
세로 스크롤링 생성 (아래에서 위로)

인자

sCid
스크롤링할 내용이 담겨있는 객체 id
iMillisec1
처음 시작 텀 (밀리초:1000=1초)
iMillisec2
실행 텀 (밀리초), 클수록 느려짐
iSpeed
한번에 움직일 px, 클수록 빨라짐
iHeight
값을 지정할경우 iHeight값만큼 이동후 iMillisec2만큼 멈추고 다시 반복

예제

기본 리스트

  • 박지성 빠진 맨유 '솔샤르 선발출전'

  • 퍼거슨 감독, 박지성 부상 공식 확인
  • 이탈리아 의회 ‘반 훌리건법’ 통과 시켜
