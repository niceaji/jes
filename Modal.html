jes.Modal

var modal = new jes.Modal(sUrl,oOptions)
이미지 또는 ifrmae(URL)을 레이어(modal) 형태로 띄움 (litebox)

인자

sUrl
이미지 url 또는 문서 url
oOptions
옵션 ( type=image|iframe,loading:false|true,opacity ... ) 예제,소스 참고
